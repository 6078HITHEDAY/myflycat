## Music out of sync

* 音频重叠播放器 - 简化版
* 需要：yad, mpv, socat


  Ubuntu/Debian: 
```bash
sudo apt install mpv yad socat bc jq
```
  Fedora: 
```bash
sudo dnf install mpv yad socat bc jq
```
  Arch:
```bash
sudo pacman -S mpv yad socat bc jq
```
